# Crowdsourced Disaster Preparedness Platform

SafetyNet is an emergency response ecosystem designed to decentralize disaster management. By leveraging crowdsourcing and Web-GIS technology, the platform empowers citizens to act as real-time sensors during crises. It facilitates a bi-directional flow of information, ensuring that emergency resources are dispatched based on high-fidelity, ground-level data and community-driven alerts.

## 🚀 Project Overview & Features
This platform aims to provide a single source of truth during emergencies.

### Core Features (Planned & In-Progress)
*   **Interactive Map:** Real-time visualization of disaster "hotspots" and resource locations using spatial data.
*   **Crowdsourcing:** Users can add new resources, update status (Open/Full), and add comments.
*   **User Accounts:** Citizens can register to contribute data and receive alerts.
*   **Intelligent Alert System:** Geo-fenced notifications to warn users based on their proximity to danger.
*   **Community Hub:** A feed for local updates and volunteer coordination.
*   **Admin Dashboard:** A dedicated dashboard for authorities to validate incoming crowdsourced data, filter misinformation, and manage resource allocation. Admins can update incident statuses in real-time to coordinate efficient rescue operations.
*   **Human Identification under Debris (CV):** Powered by a custom-trained YOLOv8 model, the system analyzes real-time imagery to detect trapped victims in complex, high-occlusion environments with 94% accuracy.
*   **Predictive AI Risk Models:** Integrated machine learning models for Flood Prediction and Earthquake Risk Assessment that analyze historical and live environmental data to provide early warning signals and risk scores for specific regions.
*   **Crowdsourced Incident Reporting:** A multi-modal reporting interface where users can submit GPS-tagged reports including descriptions and photographic evidence. This data is instantly pushed to the backend and rendered as a live "eyes-on-the-ground" layer on the global map.

## 🛠️ Tech Stack
*   **Frontend:** React, Leaflet (Maps), Tailwind CSS
*   **Backend:** Node.js, Express.js
*   **Database:** PostgreSQL with PostGIS (Geospatial Data)
*   **AI/ML & CV:** Python, YOLOv8 (Computer Vision)
*   **Mapping:** Leaflet.js / OpenStreetMap API
